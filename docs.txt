# Create React App
npx create-react-app crypto-king

# Dependencies
Install Material UI v4 & Material UI Lab:
npm install @material-ui/core @mui/lab 

Install React Router v6:
npm install react-router-dom@6

Install Axios:
npm i axios

Install React Alice Carousel:
npm i react-alice-carousel

Google Fonts:
https://fonts.google.com/specimen/Open+Sans?category=Sans+Serif#standard-styles
