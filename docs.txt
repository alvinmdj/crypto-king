# Create React App
npx create-react-app crypto-king

# Run Development
npm start

# Dependencies
Install Material UI v4 & Material UI Lab:
npm install @material-ui/core @material-ui/lab

Install React Router v6:
npm install react-router-dom@6

Install Axios:
npm i axios

Install React Alice Carousel:
npm i react-alice-carousel

Install HTML React Parser:
npm i html-react-parser

Install React Chart js 2:
npm install --save chart.js react-chartjs-2

# Firebase

Installation: npm install firebase
